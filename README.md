# 小区物业管理系统

#### 介绍
小区物业管理系统
有BUG可留言加微

#### 软件架构
SpringBoot + Mysql

#### 项目说明

1.  管理员功能模块
> 管理员登录模块
> 住户信息管理模块
> 楼道信息管理模块
> 房间信息管理模块
> 车位信息管理模块
> 缴费信息管理模块
> 报修信息管理模块
> 投诉信息管理模块
2.  住户功能模块
> 住户登录模块
> 个人主页
> 我的投诉
> 我的报修
> 我的缴费
